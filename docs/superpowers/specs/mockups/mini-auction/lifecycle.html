Mini Auction Lifecycle

End-to-end flow from creation to results

1

Create Mini League

Creator signs in, hits "Create Mini Auction." System auto-detects this weekend's matches (Sat/Sun/Mon), generates player pool ranked by fantasy points. Creator gets invite code + shareable link.

2

Lobby & Invite

Managers join via code or link. New users auto-register on join. Lobby shows who's in (2-5 managers). Creator sees "Start Auction" button (enabled when 2+ joined).

3

30s Countdown

Creator hits start. 30-second countdown with visual timer. All managers see it. No new joins after countdown begins.

4

Live Auction

Players appear one at a time, ranked by fantasy points. For each player:

a. Player card shown with stats + base price ($1M)
b. 10s timer starts. Managers bid ($1M increments) or pass
c. Timer resets on each new bid. Last 3s = audio/visual alert
d. Timer expires → sold to highest bidder (or unsold if all pass / no bids)
e. Any manager can nominate an unsold player → queued next
5

Auction Complete → Set Lineups

Auction ends when all managers have 10 players. Managers set their Day 1 lineup (7 XI + 3 bench, captain, VC). Lock time = 1 hour before first match of the day.

6

Match Days (Sat → Sun → Mon)

Each day: lineup locks before first match, existing scoring engine runs (live + post-match), daily scores shown. Before each new day, managers can adjust XI/bench/captain/VC.

7

Results & Archive

Monday matches scored → total points summed across all days → winner declared. League archived. Managers can view history but league is complete.

End Condition

Auction ends when all managers have 10 players. If the ranked list is exhausted and managers still need players, unsold players cycle back. A manager who fills 10 early is removed from bidding — auction continues for remaining managers.
